#Langkah - Langkah Install ENSP
1) Download dengan menggunakan link yang sudah disediakan
https://drive.google.com/file/d/1tNPk-Q-Bb66W4MnSm35DMG3vVypmmoaT/view?usp=share_link
2) Install dengan urutan sebagai berikut: Bebas akan tetapi ENSP diinstall paling terakhir

4) install ENSP
5) Jalankan dan gunakan Router untuk dites apakah ENSP dapat berjalan atau tidak?

   b) kalau bisa ya Alhamdulillah :v
